1 # Yet Another Config Library for C++
2 
3 ### About
4 YACL is a library for operating configs with large and complex hierarchy. Library is designed to be as simple as possible to read configs and operate on them in code.
5 
6 ### Installation
7 YACL is using cmake to build the project.
8 
9 Build example provided below
10 ```sh
11 $ mkdir build
12 $ cd build
13 $ cmake .. && cmake --build .
14 ```

20 ### Usage
21 ```cpp
22 #include <YACL/config.h>
23 int main() {
24  yacl::Setting *root = yacl::Config::parseConfigFromFile("config.yacl");
25  yacl::Config::printConfig(root);
26 
27  int var1 = *root->field("var1");
28  std::string var2 = *root->field("var2");
29  std::string var3 = *root->field("var3");
30 
31  yacl::Setting::recDeleteGraph(root);
32  return 0;
33 }
34 ```
